Este projeto consiste no desenvolvimento de um sistema de gerenciamento de estoque e vendas executado via terminal, utilizando a linguagem Python.
Estruturas de Dados Utilizadas

Lista Encadeada
Ela foi escolhida por representar uma estrutura dinâmica, sem necessidade de tamanho fixo.

Fila 
A primeira venda realizada é a primeira a ser registrada e visualizada, estrutura garante que as vendas sejam mantidas em ordem cronológica.

Pilha (LIFO)
Utilizada para implementar a funcionalidade de desfazer operações. A última operação realizada é a primeira a ser desfeita.
